Zentra CMF — мощный, гибкий и универсальный инструмент для создания сайтов любой сложности. Вы сами формируете рубрики с необходимым набором полей, а система, на основе их, строит финальный интерфейс. Никакого PHP в шаблонах — только простые теги вида: `[field:price]`, `[nav:main]`, `[block:promo]`.
